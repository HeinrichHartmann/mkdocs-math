# mkdocs-bibtex

**Version:** 4.4.0
**License:** MIT License
**Source:** https://github.com/greenape/mkdocs-bibtex

This is a vendored copy of mkdocs-bibtex bundled with the project for the following reasons:

1. **Customization** - Allows injection of citation linking logic without maintaining a fork
2. **Guaranteed availability** - No dependency on external package indexes
3. **Version control** - All modifications are tracked in git

## Modifications

Currently vendored without modifications. Future enhancements planned:
- Citation tag linking to footnote anchors
