Basic Environments

(2) Definition (\(\mathbb{R}^n\)). A Euclidean space is the vector space \(\mathbb{R}^n\) equipped with the standard inner product.

(3) Theorem. For any metric space, open sets satisfy the axioms of a topology.

(4) Proposition (Cauchy-Schwarz Inequality). For vectors \(u, v \in \mathbb{R}^n\), we have \(|u \cdot v| \leq \|u\| \|v\|\).

(5) Lemma. Every closed and bounded subset of \(\mathbb{R}^n\) is compact.

(6) Corollary. A continuous image of a compact space is compact.

Environments with Math in Content

(7) Definition (Continuity). A function \(f: X \to Y\) is continuous if for every open set \(U \subseteq Y\), the preimage \(f^{-1}(U)\) is open in \(X\).

(8) Theorem (Intermediate Value Theorem). If \(f: [a,b] \to \mathbb{R}\) is continuous and \(f(a) < 0 < f(b)\), then there exists \(c \in (a,b)\) such that \(f(c) = 0\).

Proof. Consider the set \(S = \{x \in [a,b] : f(x) \leq 0\}\). By continuity, \(S\) is closed. By the Completeness Axiom, \(\sup S\) exists.

(9) Example (Open ball in \(\mathbb{R}^2\)). The open ball of radius \(r > 0\) centered at origin is: $\(B_r(0) = \{(x, y) : x^2 + y^2 < r^2\}\)$

Complex Environments

(10) Theorem (Compact Open Topology). Let \(X\) and \(Y\) be topological spaces. The compact-open topology on \(C(X, Y)\) is generated by subbasic sets: $\(\mathcal{S} = \{\{f \in C(X, Y) : f(K) \subseteq U\} : K \subseteq X \text{ compact}, U \subseteq Y \text{ open}\}\)$

(11) Proposition (Heine-Borel). A subset \(K \subseteq \mathbb{R}^n\) is compact if and only if it is closed and bounded.

(12) Definition (Metric Space). A metric space is a pair \((X, d)\) where \(X\) is a set and \(d: X \times X \to \mathbb{R}_{\geq 0}\) satisfies: 1. \(d(x, y) = 0 \iff x = y\) 2. \(d(x, y) = d(y, x)\) 3. \(d(x, z) \leq d(x, y) + d(y, z)\)

Environments without Labels

(13) Remark. The compact-open topology is finer than the pointwise convergence topology but coarser than the uniform topology.

(14) Example. The metric \(d(x, y) = |x - y|\) makes \(\mathbb{R}\) a metric space.

(15) Note. All norms on finite-dimensional vector spaces are equivalent.

(16) Definition (\(C^k(\Omega)\), compact–open topology). The space \(C^k(\Omega)\) is equipped with the compact-open topology generated by seminorms involving derivatives up to order \(k\) on compact subsets.
